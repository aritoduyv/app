# HiddenFormField (deprecated)

Vui lòng sử dụng hàm **setHiddenField** thay cho hàm này

**Return type:** void

**Arguments:** (field)

* **field:** required, string, là tên trường data cần ẩn

**Example:**

```javascript
p.setHiddenField("ma_vt", true)
```
