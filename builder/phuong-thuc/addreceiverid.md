# addReceiverId

**Return type:** void

**Arguments**: (receiverId)

* **receiverId**: required, type string, là socket\_id của người nhận (đọc thêm [**socketFunction**](../../js/socketfunction.md))

**Example:**

```javascript
p.addReceiverId("ABCSJSJKA")
```
